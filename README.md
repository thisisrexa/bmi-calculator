## A Simple BMI Caclulator 

An app to calculate the BMI of a person. 

Created with [ViteJS](https://github.com/vitejs/vite).

## Instalation 
`npm install`

or 

`yarn install` or `yarn`

## Usage
`npm run dev`

or

`yarn dev`
